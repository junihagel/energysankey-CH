Privacy Policy

This website is a private, non-commercial website operated from Switzerland. This page explains what data (if any) is collected when you visit this website.

Website Analytics

This website uses TWIPLA (formerly Visitor Analytics) to collect anonymous usage statistics in order to understand how the website is used and to improve its content.

TWIPLA is configured to run in Maximum Privacy Mode. In this mode:

  • No cookies are used
  • No IP addresses are stored
  • No device fingerprinting is performed
  • No personal or identifiable data is collected
  • No data is shared with third parties for advertising purposes

The collected data is fully anonymous and aggregated and is used solely for statistical purposes.

Because no personal data is processed and no cookies are used, no consent banner is required.

Data Protection

This website does not collect, store, or process personal data of visitors. No user accounts, contact forms, or tracking profiles are used.

This privacy policy may be updated if the website functionality changes.
